Kingdoms

Empire-Building Strategy Game

Kingdoms is a strategy game where you lead a nation throughout history.

Dependencies

  • Boost
  • SDL
  • SDL_image
  • SDL_ttf
  • zlib
  • libpng
  • libFreeType

Download

You can download the source code in either zip or tar formats.

Windows binaries are also available:

  • Kingdoms 0.1.0

You can also clone the project with Git by running: 

$ git clone git://github.com/anttisalonen/kingdoms

Installation

Installing Kingdoms from sources:

  1. install the dependencies
  2. make
  3. (as root) make install

Installing the Windows binary

Simply unpack the zipfile into a new directory. To uninstall, remove the directory.
